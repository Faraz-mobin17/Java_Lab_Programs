# Welcome to Java

## How to compile Java Program

`C:\>javac Filename.java`

## How to run Java Program

`C:\>java Filename`

**OR**

`C:\>java Filename.java`

## How to run Applet Program

`C:\>appletviewer.exe Filename.java`
